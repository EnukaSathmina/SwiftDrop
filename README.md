## 🚀 Overview

**SwiftDrop** is a modern desktop file sharing app designed for fast local transfers over Wi-Fi.

It allows users to send and receive files between PCs and mobile devices on the same local network using a clean desktop interface, mobile browser upload/download pages, QR code connection, and a built-in shared files manager.

SwiftDrop is built as a portfolio project by **Enuka Sathmina** under **Exonix Labs**, focusing on real-world desktop app development, networking, UI design, update systems, and secure local file handling.

---

## ✨ Key Features

- ⚡ **Local Wi-Fi File Sharing**  
  Transfer files between devices on the same network without internet.

- 📱 **Mobile Browser Upload**  
  Send files from your phone to your PC using a browser upload page.

- 💻 **PC-to-Mobile Download Page**  
  Select files on your PC and let your phone download them from a local web page.

- 🔗 **QR Code Connection**  
  Scan a QR code to quickly open upload or download pages.

- 📂 **Shared Files Manager**  
  Choose exactly which files are available for other devices to download.

- 🧾 **Transfer History**  
  Track sent and received files with status, size, time, and device details.

- 🔐 **Optional PIN Protection**  
  Restrict access to upload/download pages on trusted networks.

- 🔄 **Portable Update System**  
  GitHub Releases update checking with safe updater helper support.

- 🎨 **Modern PySide6 Interface**  
  Premium dark UI with dashboard, cards, sidebar navigation, and clean controls.

# 🛡️ Security Notes

SwiftDrop is designed for trusted local networks only.

Use SwiftDrop only on Wi-Fi networks where you trust connected devices. The app exposes local upload/download pages on your LAN, so anyone on the same network may be able to access them if they know the local URL, unless PIN protection is enabled.

Security features include:

- Filename sanitization
- Path traversal protection
- Optional PIN protection
- Duplicate file auto-renaming
- Streamed file uploads
- Local-only transfer design
- No cloud storage dependency

Local IP addresses such as 192.xxx.x.x are private LAN addresses and are shown dynamically by the app.
